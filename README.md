# ☕ 100 Days of Coding – Java

Welcome to my **#100DaysOfCode** challenge! 🚀

This repository documents my journey of learning and mastering **Java Programming** by solving coding problems, implementing algorithms, and strengthening core programming concepts over the course of 100 days.

The goal of this challenge is to build a strong foundation in Java while improving problem-solving skills, coding consistency, and software development practices.

---

## 🎯 Objectives

- Learn Java from the fundamentals to advanced concepts.
- Improve logical thinking and problem-solving abilities.
- Practice writing clean and efficient Java code.
- Understand Data Structures and Algorithms (DSA).
- Develop consistency by coding every day for 100 days.
- Build a strong GitHub portfolio with daily progress.

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/pratyushkumarmishra02/100-days-of-coding-Java.git
```

2. Navigate to the project folder

```bash
cd 100-days-of-coding-Java
```

3. Open any day's folder.

4. Compile the Java file

```bash
javac FileName.java
```

5. Run the program

```bash
java FileName
```
