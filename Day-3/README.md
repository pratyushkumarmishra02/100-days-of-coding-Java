# 📝 Program Overview

## 1️⃣ Largest of Three Numbers

Determines the largest number among three integers entered by the user.

### Concepts Used

* `if-else`
* Methods
* User Input (`Scanner`)
* Conditional Statements

### Example

**Input**

```text
12
45
30
```

**Output**

```text
45 is largest.
```

---

## 2️⃣ Count Vowels and Consonants

Counts the total number of vowels and consonants present in a sentence.

### Concepts Used

* Strings
* Character Processing
* `for-each` Loop
* `Character.isLetter()`
* `indexOf()`

### Example

**Input**

```text
Hello Java
```

**Output**

```text
Total number of vowels is: 4
Total number of consonants is: 5
```

---

## 3️⃣ Reverse a String

Reverses a given string by iterating through the characters from the end to the beginning.

### Concepts Used

* Strings
* Loops
* Character Manipulation

### Example

**Input**

```text
Raja
```

**Output**

```text
ajaR
```

---

## 4️⃣ Reverse a Sentence

Completely reverses a sentence using the `StringBuilder` class.

### Concepts Used

* Strings
* `StringBuilder`
* Reverse Method

### Example

**Input**

```text
I love coding
```

**Output**

```text
gnidoc evol I
```
